<div align="center">
  <h1>Dotfiles</h1>
</div>

Arch Hyprland installation and configuration files.

Clone
```bash
git clone http://github.com/hyprland/dotfiles ~/.dotfiles
```

Run
```bash
./link.sh
./install.sh
./config.sh
```
in that order to setup the system.

Run
```bash
./wsl.sh
```
to setup WSL on W*ndows.
